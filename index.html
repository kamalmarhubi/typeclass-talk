A quick intro to type classes

The problem

Want ad-hoc polymorphism

(aka overloading)

operations that work differently on different types

Some examples

  • equality
  • hashing
  • ordering
  • stringifying

Different from parametric polymorphism

operations that work the same way on different types

        head :: [a] -> a

        fst :: (a, b) -> a

Simple example: stringifying

  • the Show class

Defining the type class

        class Show a where
          show :: a -> String

Instance declaration

        instance Show Bool where
          show True  = "True"
          show False = "False"

Default methods

        class Eq a where
          (==), (/=) :: a -> a -> Bool

          x == y = not (x /= y)
          x /= y = not (x == y)
        instance Eq Bool where
          True  == True   = True
          False == False  = True
          _     == _      = False

Using typeclass methods


    same x y =
      if x == y then "yay" else "boo"
    same :: Eq a => a -> a -> String
    same x y =
      if x == y then "yay" else "boo"

  []     == []     = True
  (x:xs) == (y:ys) = x == y && xs == ys
  _      == _      = False

overloaded constants

        class Default a where
          def :: a
        instance Default Int where
          def = 0

        instance Default Bool where
          def = False

        instance Default [a] where
          def = []
